Projects

I engineered a full-stack photo management application with a Spring Boot backend and JavaFX desktop client. It features REST endpoints for CRUD operations, multipart file uploads, and a pre-trained VGG16 deep learning model for automatic content tag generation. Its "local-first" architecture ensures user privacy by processing all image data on the local machine.

This desktop financial management application was architected using Java and JavaFX, strictly adhering to the Model-View-Controller (MVC) design pattern. I engineered the core logic for complete CRUD (Create, Read, Update, Delete) operations, enabling users to efficiently manage debts, income, and expenses.

I engineered a command-line version control system in C++ from the ground up, mimicking the core functionality of Git, including file staging, committing, and checkout. The commit history was implemented using a singly linked list, and a custom hash table with chaining was developed to store and index commit messages for efficient keyword-based searching.
